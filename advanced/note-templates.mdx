## Overview

Note templates let you create reusable text snippets for invoices and proposals. Standardize payment terms, thank-you messages, scope descriptions, and other common text to save time and maintain consistency.

## What Are Note Templates?

Templates are saved text blocks that can be inserted into:
- Invoice notes
- Proposal scope sections
- Proposal client messages
- Email templates

**Benefits:**
- **Consistency**: Same professional messaging every time
- **Time Savings**: Don't retype common text
- **Professionalism**: Well-crafted, error-free messaging
- **AI Integration**: Agents reference templates for consistent language

## Creating Note Templates

### Basic Template Creation

1. Navigate to **Settings** â†’ **Note Templates**
2. Click **Create Template**
3. Fill in template details

**Required:**
- **Name**: Template identifier (for you, not client-facing)
- **Content**: The text to reuse

**Optional:**
- **Description**: What this template is for
- **Category**: Organize by type (Payment Terms, Thank You, Scope, etc.)
- **Default**: Set as default for invoices or proposals

4. Click **Save**

### Template Types

**Payment Terms:**
```
Payment is due within 30 days of invoice date.
Late payments subject to 1.5% monthly interest.
For questions, contact accounting@yourbusiness.com.
```

**Thank You Messages:**
```
Thank you for your business! We appreciate the opportunity
to work with you and look forward to our continued partnership.
```

**Project Scope:**
```
This project includes design, development, testing, and deployment.
Two rounds of revisions included. Additional revisions billed at
standard hourly rate. Timeline: 4-6 weeks from approval.
```

**Late Payment Notice:**
```
This invoice is now overdue. Please remit payment immediately
to avoid service interruption. Contact us if you need to discuss
payment arrangements.
```

## Using Templates

### In Invoices

**To Insert Template:**
1. Create or edit invoice
2. In **Notes** field, click **Insert Template**
3. Select template from list
4. Template text inserted
5. Edit if needed for this specific invoice
6. Save invoice

**Default Templates:**
- Set default template for all new invoices
- Auto-fills notes field
- Can override per invoice

### In Proposals

**Scope Section:**
1. Create or edit proposal
2. In **Scope** field, click **Insert Template**
3. Select scope template
4. Customize for this proposal

**Client Message:**
1. In **Client Message** field
2. Click **Insert Template**
3. Select message template
4. Personalize as needed

### Default Templates

Set templates to prefill automatically:

**Invoice Default:**
1. Edit template
2. Check **Default for Invoices**
3. Save
4. All new invoices use this template

**Proposal Default:**
1. Edit template
2. Check **Default for Proposals**
3. Save
4. All new proposals use this template

<Tip>
  Default templates save the most time. Set them for your most common messaging!
</Tip>

## AI Integration

Templates enhance AI Agent capabilities:

**How AI Uses Templates:**
- References stored templates when creating invoices/proposals
- Ensures consistent client-facing language
- Maintains your brand voice
- Reduces hallucinations (uses your actual text)

**Example:**
```
User: "Create an invoice for ACME Corp for consulting"
Agent: Generates invoice with your default payment terms template
Result: Consistent, professional invoice matching your brand
```

## Managing Templates

### Template Library

**Organization:**
- Create categories: Payment, Scope, Thank You, Legal, etc.
- Search by name or content
- Sort by category or frequency of use
- Archive outdated templates

### Editing Templates

**To Edit:**
1. Open template
2. Click **Edit**
3. Update content
4. Save changes

**Impact:**
- Existing invoices/proposals NOT affected
- Only new uses of template get updated text
- Historical documents preserved

### Deleting Templates

**To Delete:**
1. Open template
2. Click **Delete**
3. Confirm deletion

**Impact:**
- Template removed from library
- Cannot be inserted into new documents
- Historical uses preserved (text already in documents stays)

<Warning>
  Only delete templates you're certain you won't need. Consider archiving instead.
</Warning>

## Best Practices

<Tip>
  **Create templates for repetitive text** - Anything you type more than 3 times deserves a template.
</Tip>

<Tip>
  **Use clear template names** - "Net 30 Payment Terms" is better than "Terms 1".
</Tip>

<Tip>
  **Keep templates professional** - These appear in client-facing documents.
</Tip>

<Tip>
  **Review and update regularly** - Update contact info, policies, pricing as they change.
</Tip>

<Tip>
  **Set defaults for common cases** - Default invoice and proposal templates save the most time.
</Tip>

<Tip>
  **Categorize templates** - Helps find the right template quickly.
</Tip>

## Common Template Examples

### Payment Terms

**Net 30 Standard:**
```
Payment due within 30 days of invoice date. We accept credit cards,
bank transfers, and checks. Late payments subject to 1.5% monthly interest.
```

**Net 15 with Online Payment:**
```
Payment due within 15 days. Pay securely online with credit card using
the "Pay Now" button above, or send payment to [bank details].
```

### Thank You Messages

**Standard Thank You:**
```
Thank you for your business! We appreciate your trust and look forward
to continuing to serve you.
```

**First-Time Client:**
```
Welcome! Thank you for choosing us for your project. We're excited to
work with you and committed to exceeding your expectations.
```

### Scope Templates

**Website Development:**
```
Includes: Custom design, responsive development, CMS integration, SEO
optimization, 2 revision rounds. Excludes: Content creation, photography,
ongoing maintenance. Timeline: 6-8 weeks from approval.
```

**Monthly Consulting Retainer:**
```
10 hours per month of consulting services. Unused hours do not roll over.
Additional hours billed at $[rate]/hour. Includes email/phone support
and one monthly strategy session.
```

### Legal/Terms

**Standard Terms:**
```
Work begins upon receipt of signed agreement and 50% deposit. Final
payment due upon completion. Client owns final deliverables upon full
payment. [Your Business] retains right to use work in portfolio.
```
