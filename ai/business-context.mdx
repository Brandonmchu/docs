## Overview

The Business Context Workspace lets you customize how the Every Agent understands your business. Add context through organization prompts, file uploads, Drive links, and URLs so the Agent provides more relevant, accurate assistance.

**What You Can Customize:**
- Organization prompt (how Agent sees your business)
- Reference files (documents Agent can access)
- Drive links (Google Drive documents)
- URLs (web pages for reference)
- Trigger business analysis for insights

## Accessing Business Context

### Global Chat Dialog

**Location:** Bottom-right chat icon → Click Settings (gear icon) in chat header → Navigate to context and configuration tabs

**Features:**
- Edit organization prompt
- Upload reference files
- Add Drive/web links
- Configure scheduled tasks
- Access business analysis

**Always Accessible:**
- Available from any page
- Persists across sessions
- Syncs with main application

<Tip>
  Keep business context updated as your business evolves. Fresh context = better Agent responses!
</Tip>

## Organization Prompt

### What is Organization Prompt?

**Custom Instructions for Agent:**
- Describes your business
- Explains your services
- Sets tone and style
- Defines priorities
- Provides context

**How Agent Uses It:**
- References before every response
- Shapes recommendations
- Influences invoice creation
- Guides email drafting
- Informs analysis

### Editing Organization Prompt

**To Edit:**
1. Open chat dialog
2. Click context/settings icon
3. Find **Organization Prompt** section
4. Click **Edit**
5. Update text
6. Save changes

**What to Include:**
- **Business Description**: What you do
- **Target Clients**: Who you serve
- **Services Offered**: Main offerings
- **Business Values**: How you work
- **Communication Style**: Tone preferences
- **Special Instructions**: Unique requirements

**Example Prompt:**
```
We are a boutique digital marketing agency specializing in
e-commerce brands. Our clients are typically $1M-$10M revenue
online retailers looking to scale.

Services:
- Paid advertising management (Facebook, Google)
- Email marketing campaigns
- Conversion rate optimization
- Monthly retainers ($3K-$10K/month)

Communication style: Professional but friendly, data-driven,
results-focused. Always emphasize ROI in client communications.

Invoice preferences: Net 30 terms, detailed line items showing
hours and deliverables, always include next month's strategy
preview in notes.
```

<Note>
  The more specific your organization prompt, the better the Agent understands your business!
</Note>

## File & Link References

### Uploading Reference Files

**Supported Files:**
- PDFs (contracts, proposals, guidelines)
- Word documents (templates, policies)
- Text files (notes, procedures)
- Spreadsheets (rate cards, pricing)
- Maximum 5MB per file

**To Upload:**
1. Open business context settings
2. Find **Reference Files** section
3. Click **Upload File** or drag and drop
4. File uploaded and indexed
5. Agent can now reference it

**How Agent Uses Files:**
- Searches content when relevant
- References data in responses
- Cites information from files
- Stays consistent with policies

**Use Cases:**
- Service agreements (standard terms)
- Rate cards (pricing reference)
- Brand guidelines (messaging consistency)
- Project templates (standard deliverables)
- FAQs (common client questions)

<Tip>
  Upload your standard service agreement so Agent can reference terms when creating invoices!
</Tip>

### Linking Google Drive Documents

**Drive Integration:**
- Link Google Drive files
- Agent accesses with your permissions
- Always up-to-date (links to live document)
- No upload size limits

**To Add Drive Link:**
1. Open business context settings
2. Click **Add Drive Document**
3. Search or paste Drive link
4. Select document
5. Link added to context

**Benefits Over Upload:**
- Live documents (always current)
- Collaborative docs (team updates reflected)
- Larger files (no size limit)
- Organized in Drive (single source of truth)

### Adding Web URLs

**Reference Websites:**
- Link to your website
- Industry resources
- Client portals
- Knowledge bases
- Pricing pages

**To Add URL:**
1. Open business context settings
2. Find **Web References** section
3. Click **Add URL**
4. Paste URL
5. Add description (what it's for)
6. Save

**Agent Access:**
- Fetches page content
- References information
- Stays current with your site
- Uses for pricing, services, policies

**Example URLs:**
- Your services page (for service descriptions)
- Your pricing page (for rate reference)
- Industry best practices (for recommendations)
- Client success stories (for credibility)

<Tip>
  Add your website's services and pricing pages so Agent knows your current offerings!
</Tip>

## Business Insights

Every Agent can analyze your business data to provide insights on demand. These insights appear directly in agent chat responses when you ask questions or trigger a comprehensive analysis.

**Agent Analyzes Your Data:**
- Revenue patterns
- Client behavior
- Payment trends
- Service performance
- Growth opportunities

**Insight Types:**
- **Revenue Insights**: Top clients, trending services
- **Risk Insights**: Slow payers, overdue invoices
- **Opportunity Insights**: Upsell potential, new services
- **Operational Insights**: Efficiency improvements
- **Financial Insights**: Cash flow, profitability

**To Get Insights:**
- Click the "Analyze Business" button in the Settings panel, or
- Ask Every Agent questions like:
  - "How's my business doing?"
  - "What are my revenue trends?"
  - "Which clients should I focus on?"
  - "Run a business analysis for me"

**Example Insights:**
```
"Your top 3 clients generate 60% of revenue. Consider
diversifying client base to reduce risk."

"Average days to payment: 45 days. Your terms are Net 30.
Consider sending earlier reminders or offering early payment
discounts."

"Consulting services have 85% margin vs 40% for implementation.
Consider shifting focus to higher-margin consulting."
```

<Tip>
  Insights are generated on-demand based on your current business data. Trigger analysis when you want fresh insights!
</Tip>

## Customization Tips

### Effective Organization Prompts

**Be Specific:**
- ❌ "We're a consulting firm"
- ✅ "We're a healthcare IT consulting firm specializing in EHR implementations for hospitals"

**Include Examples:**
```
Our typical project: 3-month EHR implementation,
$150K-$300K, includes training and go-live support.
```

**Set Tone:**
```
Communication: Professional medical environment,
HIPAA-compliant language, emphasize data security.
```

**Define Priorities:**
```
Always prioritize: Data security, compliance, thorough
documentation. Never compromise on security for speed.
```

## Troubleshooting

### Agent Not Using Context

**Check:**
- Context saved properly
- Files uploaded successfully
- Links accessible
- Prompt not too vague

**Solution:**
- Re-save organization prompt
- Re-upload files
- Verify link permissions (Drive)
- Make prompt more specific

### File Upload Failed

**Check:**
- File size under 5MB
- Supported format
- Internet connection
- Storage quota

**Solution:**
- Compress large files
- Convert to PDF
- Use Drive link instead
- Contact support

### Drive Link Not Working

**Verify:**
- File permissions (shared with your account)
- Link not expired
- File not deleted
- Google Drive integration connected

**Solution:**
- Check file sharing settings
- Reconnect Google Drive
- Use direct file upload instead
