## Overview

Business Context is the workspace where you teach Every Agent how your business works. It combines written context, uploaded files, Google Drive references, and web links so the agent can answer questions and take action with better business-specific judgment.

<Card title="Open Business Context" icon="brain" href="https://every.ai/agent/settings/context" horizontal>
  Manage your agent's business context
</Card>

**Use Business Context for:**
- Business description and positioning
- Services, products, and pricing logic
- Target clients and ideal customer profile
- Internal operating rules or constraints
- Supporting files and reference links

## Accessing Business Context

Go to [**Agent Settings → Context**](https://every.ai/agent/settings/context).

This page is the current home for:
- Your editable business context
- Business context generation from workspace data
- Uploaded reference files
- Google Drive references
- Web links

<Note>
  Older docs may refer to an "organization prompt." That content now lives inside the broader Business Context workspace.
</Note>

## What Business Context Does

Business Context helps the agent understand:
- What your business sells
- How you describe your work
- Which clients you serve
- What rules or preferences it should follow
- Which source documents it should use as references

This improves:
- Invoice and proposal drafting
- Business Q&A
- Data analysis
- Follow-up suggestions
- Context-aware recommendations

## Written Business Context

The written context area is where you describe your business directly.

**Good things to include:**
- What your company does
- Main services and products
- Target industries or client types
- Pricing philosophy
- Delivery model and process
- Important rules the agent should follow

**Example:**

```text
We are a boutique product strategy firm working with B2B SaaS companies.
Our work is usually sold as discovery sprints, roadmap workshops, and
fractional product leadership retainers.

Typical engagements are $5K-$20K. Keep communication concise, confident,
and practical. We prefer clear scope, milestone-based invoicing, and
Net 15 terms for smaller clients.
```

<Tip>
  Write context the way you would brief a new operator joining your company. That level of specificity works well.
</Tip>

## Build Context From Your Workspace

Business Context can also be generated from your existing workspace data.

This uses your current data to build or refresh a broader business summary from things like:
- Clients
- Financial activity
- Communication patterns
- Workspace structure

Use **Build Context** when:
- You are setting things up for the first time
- Your business has changed materially
- You want the agent to refresh its broader understanding from recent activity

<Warning>
  Building context can replace the current generated narrative. Review the result after it finishes and edit anything that needs tightening.
</Warning>

## Files & Link References

Business Context is not limited to typed instructions. You can also add files and links the agent should reference.

### Uploaded Files

Upload files directly in Business Context.

**Common file types:**
- PDF contracts
- Word proposals or SOW templates
- Excel rate cards or planning sheets
- CSV data exports
- Images and screenshots
- Text notes

**Good uploads:**
- Standard agreements
- Pricing sheets
- Brand guidelines
- Deliverable templates
- Internal playbooks

### Google Drive References

If you use Google Drive, you can attach live Drive documents so the agent references the current version instead of a static upload.

**Good for:**
- Living docs
- Shared templates
- Team-maintained policies
- Frequently updated reference material

### Web Links

You can also add URLs the agent should reference.

**Useful examples:**
- Your marketing site
- Pricing page
- Services page
- Support docs
- Client onboarding resources

<Tip>
  Prefer live Drive docs or URLs when the source changes often. Prefer uploads when you want a fixed snapshot.
</Tip>

## How the Agent Uses Files and Links

The agent can:
- Search them when relevant
- Pull details into responses
- Reuse consistent language
- Reference terms, pricing, and process details

Examples:

```text
"Use my service agreement when drafting this proposal"
"Check the pricing sheet before creating that invoice"
"Read the onboarding guide and summarize our implementation steps"
```

## Business Context vs Voice & Tone

These are separate on purpose:

| Setting | What It Controls |
| - | - |
| **Business Context** | What the agent knows about your business |
| **Voice & Tone** | How the agent writes when speaking on your behalf |

Use Business Context for business facts, rules, and operating details.

Use [Voice & Tone](/ai/voice-and-tone) for writing style, tone, and audience-specific communication.

## Best Practices

<Tip>
  Keep the written context practical. Focus on what the agent should know to make better decisions.
</Tip>

<Tip>
  Add source documents for anything you want the agent to reference consistently, such as pricing, terms, or messaging.
</Tip>

<Tip>
  Refresh the context when your positioning, pricing, or client mix changes significantly.
</Tip>

<Tip>
  Review generated context after a rebuild. Treat it as a draft to refine, not untouchable output.
</Tip>

## Troubleshooting

### Agent Not Using My Context

**Check:**
- Your changes were saved
- Files uploaded successfully
- Links are still accessible
- The context is specific enough to be actionable

**Try this:**
- Tighten vague sections
- Re-upload important files
- Fix broken links or permissions
- Ask the agent a task that clearly depends on the context you added

### File Upload Failed

**Check:**
- File format is supported
- Internet connection is stable
- The file is not corrupted

**Try this:**
- Compress or convert the file
- Upload a PDF instead of a more complex format when possible
- Use a Drive link if the file works better as a live document

### Drive Link Not Working

**Check:**
- Google Workspace is connected
- The file still exists
- The account permissions are correct

**Try this:**
- Reconnect Google
- Re-add the Drive file
- Use a direct upload if you need a fixed copy instead
