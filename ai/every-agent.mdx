## Overview

Every Agent is your AI-powered business assistant. Chat naturally to create invoices, import data, analyze finances, send reminders, and get business insights - all through simple conversation.

**What Every Agent Can Do:**
- Create and manage invoices
- Import and analyze CSV data
- Generate business insights
- Send email reminders
- Track expenses and receipts
- Answer questions about your business
- Automate repetitive tasks
- Provide onboarding guidance

## How to Use Every Agent

### Accessing the Agent

**Global Chat Dialog:**
- Click chat icon (bottom right)
- Available on all pages
- Persistent conversation history
- Context-aware responses

**Chat Interface:**
- Type your request naturally
- Agent responds in real-time
- See actions being performed
- Review and approve actions

<Tip>
  Just describe what you want in plain English. The Agent understands natural language!
</Tip>

### Conversational Interface

**No Special Commands:**
- No need to memorize syntax
- Talk like you're texting a colleague
- Agent understands context
- Ask follow-up questions naturally

**Examples:**
```
You: "Create an invoice for ACME Corp for $5,000"
Agent: Creates invoice, shows preview, confirms

You: "Actually, make it $5,500"
Agent: Updates amount, shows new total

You: "Send it"
Agent: Sends invoice, provides tracking link
```

## Agent Turn Undo

### Undo Last Action

**If Agent Made Mistake:**
```
"Undo that"
"Reverse the last action"
"That was wrong, undo it"
```

**Agent Will:**
- Identify last action set
- Reverse changes
- Restore previous state
- Confirm undo complete

**What Can Be Undone:**
- Invoice creation
- Client creation
- Expense logging
- Data imports (partial)
- Email sends (can't unsend, but logs corrected)

**Limitations:**
- Can't undo sent emails (already delivered)
- Can't undo completed payments
- One level of undo (not multi-level)

<Warning>
  Undo works for most actions, but sent emails and processed payments cannot be reversed!
</Warning>

4. Review extracted data
5. Approve or adjust
6. Expense logged with receipt

- Amount: 90%+ accurate
- Date: 90%+ accurate
- Category: 70%+ (verify recommended)

## Troubleshooting

### Agent Not Understanding Request

**Make it More Specific:**
- Include client name
- Specify amounts
- Mention invoice numbers
- Add context

**Example:**
- Vague: "Make an invoice"
- Better: "Create invoice for ACME Corp for $5,000"

### Agent Made Mistake

**Undo and Clarify:**
```
"Undo that"
[Agent undoes]
"Create invoice for ACME Corp (not XYZ Corp) for $5,000"
```

### Action Needs Approval

**Agent Awaits Confirmation:**
- Review proposed action
- Type "yes" or "approve" to proceed
- Or provide corrections
- Agent adjusts and re-proposes

### Tool or Integration Unavailable

**Agent Will Report:**
- "Gmail integration not connected"
- "Need to connect Stripe first"
- "Client doesn't exist, should I create it?"

**Solution:**
- Set up required integration
- Create missing prerequisites
- Agent guides you through setup
