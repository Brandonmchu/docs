## Overview

<Card title="Open Export" icon="file-export" href="https://every.ai/data/export-invoices" horizontal>
  Export your data to CSV
</Card>

Export invoice and expense data to CSV for accounting software or your own reporting workflows. Every includes preset formats for Xero and Wave plus a fully customizable export mode.

**Key capabilities:**
- Preset formats for **Xero** and **Wave**
- **Custom** exports with configurable columns
- Invoice and expense export modes
- Date range and status filtering
- Live preview before download
- Reusable templates for recurring exports
- Fork preset formats into custom ones

## Quick Start

**Invoice export:** Data → Export Invoices → Choose preset → Set filters → Preview → Download CSV

**Expense export:** Data → Export Expenses → Set filters → Preview → Download CSV

**Custom export:** Start from Xero or Wave, click **Customize**, then add/remove/reorder columns and save as a template if needed

## Export Types

### Invoice Export

Invoice export supports flexible row models and field selection.

**Row models:**
- **Invoice Summary**: One row per invoice with totals
- **Line Item Detail**: One row per line item for more detailed accounting imports

**Available field groups:**
- **Invoice**: Number, issue date, due date, status, payment status, currency, total, reference, notes
- **Client**: Name, ID
- **Line Items**: Name, description, quantity, rate, line total, discount, tax fields
- **Computed**: Customer email, net amount, tax amount, gross amount, amount paid, balance due, paid date

### Expense Export

Expense export is designed for bookkeeping and reimbursement workflows.

**Available field groups:**
- **Expense**: Date, amount, description, category, merchant, receipt URL
- **Client**: Name and ID for billable expenses
- **Billing**: Whether the expense has been invoiced and linked invoice details
- **Metadata**: Created date and related internal IDs

## Preset Formats

### Xero

Xero export is optimized for invoice import into Xero.

**Format details:**
- Xero-compatible invoice columns
- Date format: `DD/MM/YYYY`
- Includes `AccountCode`
- Best for draft invoice imports into Xero

**Typical import flow:**
1. Export from Every in **Xero** format
2. In Xero, go to **Business → Invoices → Import**
3. Upload the CSV
4. Review field mapping and imported drafts

<Note>
  Xero imports usually create drafts. Review them in Xero before approving or sending.
</Note>

### Wave

Wave export is designed for Wave Connect workflows.

**Format details:**
- Columns prepared for Wave Connect
- Date format: `YYYY-MM-DD`
- Includes customer, invoice number, line item description, quantity, price, amount, and tax fields

**Typical import flow:**
1. Export from Every in **Wave** format
2. Open Google Sheets with Wave Connect installed
3. Use Wave Connect to prepare and validate the data
4. Upload the prepared sheet to Wave

<Warning>
  Wave import depends on Wave Connect. There is no direct CSV upload flow in Every itself.
</Warning>

### Custom

Custom export gives you full control over the columns and headers.

**Best for:**
- Accountant-specific exports
- Internal spreadsheet analysis
- Systems that need different headers
- Specialized reporting or migration work

## Filtering Options

### Date Range

Use presets such as:
- This Month
- Last Month
- This Quarter
- Year to Date
- Last Year
- Custom range

### Invoice Filters

Filter invoice exports by:
- **Date mode**: issue date, due date, or paid date
- **Invoice status**: draft, issued, paid, overdue, void
- **Payment status**: unpaid, partially paid, fully paid
- **Client**

### Expense Filters

Filter expense exports by:
- Expense date
- Billing status
- Category
- Archived vs active records

## Customizing Exports

### Fork a Preset

Start with Xero or Wave, then customize it:

1. Select a preset
2. Click **Customize**
3. Add, remove, or reorder columns
4. Rename headers if needed
5. Save as a template for future reuse

<Tip>
  Forking is the fastest way to build a custom export that is close to Xero or Wave but needs a few extra fields.
</Tip>

### Field Selector

When customizing columns:
- Choose fields from grouped categories
- Reorder them in the export column list
- Rename headers to match your target system
- Remove anything you do not need

### Templates

Save recurring export setups so your team can reuse them.

**Templates can store:**
- Export type
- Preset/custom column layout
- Filters
- Date mode

**Examples:**
- Monthly accountant export
- Quarter-end revenue detail
- Unbilled expense review

## Preview & Download

Before downloading, Every shows a preview so you can verify:
- Row structure
- Headers
- Filter results
- Formatting

If the preview looks wrong, adjust filters or columns before generating the CSV.

<Tip>
  Preview first when using line-item detail exports. They can produce many more rows than invoice summary mode.
</Tip>

## Best Practices

<Tip>
  Use presets for standard accounting flows and custom templates for everything else.
</Tip>

<Tip>
  Pick the right date mode for your accountant. Cash-basis reporting often wants paid date; accrual workflows often want issue date.
</Tip>

<Tip>
  Save templates for recurring exports instead of rebuilding them every month.
</Tip>

## Troubleshooting

**Export missing expected invoices:** Check date mode, status filters, and client filters.

**Too many rows in CSV:** You may be using line-item detail instead of invoice summary.

**Wave import issues:** Validate the data through Wave Connect and make sure customer names match what Wave expects.

**Xero import issues:** Verify required columns are present and dates are in the expected Xero format.
