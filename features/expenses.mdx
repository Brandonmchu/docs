## Overview

Every AI's expense management helps you track business expenses for tax purposes and bill them back to clients. Upload receipts, let AI extract details, categorize expenses, and seamlessly add billable expenses to client invoices.

**Key Features:**
- Receipt scanning with AI extraction
- Expense categorization for tax reporting
- Client billback capabilities
- Receipt attachment and storage
- Integration with invoices
- Expense reporting and analytics

## What Are Expenses?

Expenses are business costs you incur:

**Types:**
- **Billable to Clients**: Expenses you'll invoice to clients (travel, materials, etc.)
- **Business Operating Expenses**: General business costs (software, equipment, etc.)
- **Tax-Deductible**: Expenses you'll claim on taxes

**Common Expenses:**
- Office supplies
- Software subscriptions
- Travel and meals
- Client materials
- Equipment purchases
- Professional services
- Advertising and marketing

## Creating Expenses

### Basic Expense Entry

1. Navigate to **Expenses**
2. Click **Add Expense** or **+** button
3. Fill in expense details

**Required Fields:**
- **Merchant**: Who you paid
- **Amount**: How much you spent
- **Date**: When expense occurred

**Optional Fields:**
- **Category**: Type of expense
- **Description**: Additional details
- **Receipt**: Upload receipt image/PDF
- **Client**: If billing back to client
- **Invoice**: Link to invoice (for billable expenses)
- **Payment Method**: How you paid
- **Notes**: Additional information

4. Click **Save**

<Tip>
  Upload receipts for all expenses! They're required for tax deductions and audit protection.
</Tip>

### Receipt Scanning with AI

Upload receipts and let AI extract details automatically:

**How Receipt Scanning Works:**

1. Click **Add Expense**
2. Click **Upload Receipt** or drag & drop image
3. AI analyzes receipt and extracts:
   - **Merchant name**
   - **Amount** (total)
   - **Date** of transaction
   - **Category** (best guess)
   - **Payment method** (if visible)
4. Review extracted data
5. Adjust if needed
6. Save expense

**Supported Receipt Formats:**
- Images: JPG, PNG, HEIC
- Documents: PDF
- Maximum size: 5MB per file

**AI Accuracy:**
- Merchant: 95%+ accurate
- Amount: 90%+ accurate
- Date: 90%+ accurate
- Category: 70%+ accurate (requires review)

<Note>
  Always review AI-extracted data before saving. The AI is very good but not perfect!
</Note>

## Billable Expenses

Bill expenses back to clients:

### Marking Expenses as Billable

**When Creating Expense:**
1. Add expense details
2. Check **Billable to Client**
3. Select **Client** from dropdown
4. Save expense

**Existing Expense:**
1. Edit expense
2. Check **Billable to Client**
3. Select client
4. Save changes

**Billable Expense Indicators:**
- Marked as "Billable" in expense list
- Client name shown
- Shows "Unbilled" until added to invoice
- Can be filtered by billable status

### Adding Expenses to Invoices

Convert billable expenses to invoice line items:

**To Bill Expenses:**

1. Create invoice for client
2. Click **Add from Expenses**
3. See list of unbilled expenses for this client
4. Select expenses to include
5. Click **Add Selected**
6. Expenses become invoice line items:
   - Description: Expense description
   - Amount: Expense amount
   - Marked as "Billed"

**Example:**
```
Expense: Client meeting travel
Amount: $145.50
Category: Travel

Becomes Invoice Line Item:
Description: Client meeting travel
Amount: $145.50
Quantity: 1
```

**Markup Options:**
- Add percentage markup (e.g., add 10% to expenses)
- Round to nearest $5 or $10
- Add flat handling fee

<Tip>
  Clearly communicate your expense policy with clients upfront. Some clients expect expenses at cost, others accept markup.
</Tip>

### Editing Expenses

**To Edit:**
1. Open expense
2. Click **Edit**
3. Change any field
4. Save changes

**Editable Fields:**
- All fields can be edited anytime
- Exception: Billed expenses linked to issued invoices (limited editing)

### Deleting Expenses

**To Delete:**
1. Open expense
2. Click **Delete**
3. Confirm deletion

**Restrictions:**
- Can delete unbilled expenses anytime
- Cannot delete billed expenses linked to issued invoices
- Remove from invoice first if needed

<Warning>
  Keep expense records for tax purposes. Only delete expenses recorded in error.
</Warning>

**Uploading Receipts:**

2. Click **Upload Receipt**
3. Select image or PDF

5. Visible in expense details

**Receipt Storage:**
- Securely stored in cloud
- Accessible anytime
- Downloadable
- Signed URLs for security
- Backed up automatically

**Multiple Receipts:**
- Some expenses have multiple receipts
- Upload all relevant documentation
- All stored with expense

**Missing Receipts:**
- Add expense without receipt if needed
- Upload receipt later when found
- Note in description if receipt lost

## Troubleshooting

### Receipt Upload Failed

**Check:**
- File size under 5MB
- Supported format (JPG, PNG, PDF)
- Internet connection stable
- Browser allows file uploads

**Solution:**
- Compress large images
- Convert to supported format
- Try different browser
- Contact support if persists

### AI Extracted Wrong Amount

**Common Issue:** AI reads tax or subtotal instead of total

**Solution:**
- Always review AI-extracted data
- Correct amount before saving
- Look for "Total" or "Amount Due" on receipt
- Report incorrect extraction to improve AI

### Can't Edit Billed Expense

**Reason:** Expense linked to issued invoice

**Solution:**
- Edit limited to prevent invoice inconsistency
- Remove from invoice first (if invoice is draft)
- Or accept current values to maintain invoice accuracy

### Expense Not Appearing in "Add to Invoice"

**Check:**
- Expense marked as billable
- Expense assigned to correct client
- Expense not already billed
- Expense exists (not deleted)

**Solution:**
- Verify expense is billable
- Check client assignment matches invoice
- Look in billed expenses filter
