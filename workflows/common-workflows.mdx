## Overview

This guide walks through complete workflows for common business scenarios. Follow these proven patterns to get the most out of Every AI.

## Complete New Business Setup

### Getting Started from Scratch

**Week 1: Foundation**

1. **Organization Setup** (Day 1)
   - Complete business profile (Settings â†’ Organization)
   - Upload logo and set brand colors
   - Configure default currency and due days
   - Set sales tax rate
   - Add invoice note template

2. **Payment Processing** (Day 1-2)
   - Connect Stripe account
   - Complete Stripe onboarding
   - Verify bank account
   - Test payment processing

3. **Integrations** (Day 2-3)
   - Connect Google Workspace (optional)
   - Connect HubSpot CRM (if applicable)
   - Import Google Contacts
   - Test integrations

4. **Add Clients** (Day 3-4)
   - Import from Google Contacts
   - Or manually create top 5-10 clients
   - Use client enrichment for business clients
   - Add CC email addresses

5. **Create Offerings** (Day 4-5)
   - List all services/products
   - Create offering for each
   - Set standard rates
   - Assign tax rates
   - Organize by category

**Week 2: First Invoice**

6. **Create First Invoice**
   - Select client
   - Add line items from offerings
   - Review branding and appearance
   - Send test invoice to yourself
   - Send real invoice to client

7. **Track and Follow Up**
   - Monitor email delivery
   - Note when client opens invoice
   - Record payment when received
   - Celebrate first payment!

<Tip>
  Don't try to do everything at once. Follow this week-by-week plan for smooth onboarding!
</Tip>

## Service Business Workflow

### For Consultants, Developers, Designers, etc.

**Monthly Cycle:**

**Week 1-4: Work & Track**
1. Track time using timer widget
   - Start timer when beginning work
   - Select client
   - Add description of work
   - Stop timer when done
2. Log expenses as incurred
   - Upload receipts immediately
   - Mark billable to client
   - Categorize for taxes

**End of Month: Invoicing**
3. Review unbilled time
   - Filter by client
   - Verify all hours tracked
   - Check rates accurate
4. Create monthly invoice
   - Select client
   - Add unbilled time records
   - Add unbilled expenses
   - Review total
5. Add project summary
   - Write summary of month's work
   - Highlight key achievements
   - Attach deliverables if applicable
6. Send invoice
   - Send via Every AI
   - Track email open
   - Note in calendar for follow-up

**Follow-Up:**
7. Day 7: Check if client opened email
8. Day 15: Send friendly reminder if unpaid
9. Day 25: Send urgent reminder if still unpaid
10. Record payment when received

**Example Month:**
```
Week 1: 12 hours @ $150/hr = $1,800
Week 2: 15 hours @ $150/hr = $2,250
Week 3: 10 hours @ $150/hr = $1,500
Week 4: 13 hours @ $150/hr = $1,950

Total Time: 50 hours = $7,500
Expenses: $250 (travel, software)
Invoice Total: $7,750
```

## Product Business Workflow

### For Physical or Digital Product Sellers

**Per-Order Process:**

1. **Receive Order**
   - Customer places order
   - Confirm inventory/availability

2. **Create Invoice**
   - Select/create client
   - Add products from offerings
   - Apply discounts if applicable
   - Calculate shipping (if applicable)
   - Calculate tax

3. **Send Invoice**
   - Email invoice immediately
   - Include payment link
   - Set short due date (Net 7 or Net 15)

4. **Track Payment**
   - Monitor for online payment
   - Or await manual payment notification

5. **Fulfill Order**
   - Ship product when paid
   - Or deliver digital product
   - Send tracking/access info

6. **Record Completion**
   - Update client notes
   - Archive invoice if desired
   - Request review/feedback

**Bulk/Wholesale Workflow:**
- Create standing offerings for products
- Use automations for regular orders
- Apply volume discounts
- Net 30 terms common for B2B

## Recurring Client Workflow

### For Monthly Retainers, Subscriptions, etc.

**Setup (One-Time):**

1. **Define Retainer Terms**
   - Monthly fee amount
   - Included services/hours
   - Overage rates
   - Contract length

2. **Create Automation**
   - Navigate to Automations
   - Create new automation
   - Select client
   - Set schedule: Monthly, 1st of month
   - Add line items (retainer fee)
   - Action: "Create as Draft" initially
   - Save automation

3. **Test Automation**
   - Enable test mode
   - Generate test invoice
   - Review formatting and amounts
   - Verify branding
   - Disable test mode when satisfied

4. **Activate Automation**
   - Set status to Active
   - Verify next run date correct

**Ongoing (Monthly):**

5. **Review Generated Invoice**
   - Check draft invoice on 1st of month
   - Add any overages or extras
   - Adjust if needed
   - Send invoice

6. **Track Hours** (if hours included)
   - Monitor hours used vs allocated
   - Notify client as approaching limit
   - Invoice overages separately or adjust next month

7. **Monitor Payments**
   - Payment typically arrives 5-7 days after send
   - Follow up if delayed
   - Record manual payments if not Stripe

**Annual Review:**
8. Evaluate retainer performance
9. Adjust rates if needed
10. Update automation with new rate
11. Communicate changes to client in advance

<Tip>
  Start automations with "Create as Draft" until you trust them, then switch to "Send Immediately" for full automation!
</Tip>

## Proposal to Invoice Workflow

### From Estimate to Payment

**Discovery & Proposal:**

1. **Initial Client Meeting**
   - Understand needs
   - Define scope
   - Discuss budget
   - Set expectations

2. **Create Proposal**
   - Select client
   - Add detailed scope
   - Include deliverables
   - Add line items with pricing
   - Attach examples/portfolio
   - Write personalized client message
   - Set expiration date (7-14 days)

3. **Send Proposal**
   - Email proposal
   - Track email open
   - Follow up in 3-5 days

4. **Negotiation** (if needed)
   - Discuss concerns
   - Adjust scope or pricing
   - Duplicate proposal with changes
   - Resend revised version

**Approval & Work:**

5. **Receive Approval**
   - Client approves proposal
   - Approval captured with signature
   - Work can begin

6. **Convert to Invoice**
   - Click "Convert to Invoice"
   - All line items copy over
   - Set issue and due dates
   - Add any deposit terms if applicable

7. **Send Invoice**
   - For deposit or full payment
   - Client pays
   - Work proceeds

**Completion:**

8. **Complete Work**
   - Deliver all defined deliverables
   - Meet acceptance criteria

9. **Final Invoice** (if deposit model)
   - Create invoice for remaining balance
   - Reference original proposal
   - Send final invoice

10. **Collect Payment**
    - Monitor payment
    - Send thank you
    - Request testimonial/referral

## Expense Billback Workflow

### Billing Client Expenses

**During Project:**

1. **Incur Expense**
   - Business expense related to client project
   - Examples: Travel, materials, software, supplies

2. **Log Immediately**
   - Take photo of receipt
   - Upload to Every AI: "Log this expense"
   - Agent extracts details
   - Mark as "Billable to [Client]"
   - Save with receipt attached

3. **Categorize for Tracking**
   - Assign category (Travel, Materials, etc.)
   - Add project notes if tracking per-project

**Billing Cycle:**

4. **Review Unbilled Expenses**
   - Filter expenses by client
   - Show only "Unbilled" expenses
   - Verify all legitimate billable expenses included

5. **Add to Invoice**
   - Create invoice or open existing draft
   - Click "Add from Expenses"
   - Select all billable expenses for this client
   - Expenses become line items

6. **Apply Markup** (if policy)
   - Some businesses add markup to expenses (10-20%)
   - Adjust line item amounts
   - Or add note explaining: "Reimbursable expenses at cost"

7. **Send Invoice**
   - Expenses marked as "Billed"
   - Client pays invoice including expenses
   - You're reimbursed

**Best Practices:**
- Log expenses immediately (don't wait)
- Always keep receipts
- Be transparent about expense policy
- Communicate large expenses before incurring

## Monthly Close Workflow

### End of Month Routine

**Days 25-30: Preparation**

1. **Review Month Activity**
   - Total invoices sent
   - Total payments received
   - Outstanding balance
   - Overdue invoices

2. **Follow Up on Late Payments**
   - Identify overdue invoices (filter by status)
   - Send reminders to slow payers
   - Call high-value overdue clients

3. **Finish Drafts**
   - Review all draft invoices
   - Complete and send pending invoices
   - Don't carry drafts into new month

4. **Bill Recurring Clients**
   - Review automations (if not auto-sending)
   - Trigger manual recurring invoices
   - Confirm all regulars billed

**Day 1-5: New Month Start**

5. **Generate Monthly Reports**
   - Revenue report (last month)
   - Outstanding invoices report
   - Expense summary (for accounting)
   - Tax report (if quarterly filer)

6. **Update Forecasts**
   - Expected revenue this month
   - Invoices in pipeline
   - Cash flow projection

7. **Set Monthly Goals**
   - Revenue target
   - Number of new clients
   - Payment collection goals

8. **Export for Accounting**
   - Export invoice data
   - Export expense data
   - Send to accountant
   - File for records

## Tips for Each Workflow

<Tip>
  **New Business Setup**: Don't skip the foundation. Proper setup saves time every day for years to come.
</Tip>

<Tip>
  **Service Business**: Track time religiously. Memory fades fast, so log work immediately.
</Tip>

<Tip>
  **Product Business**: Automate where possible. Use offerings and automations to reduce manual work.
</Tip>

<Tip>
  **Recurring Clients**: Test automations thoroughly before going live. One mistake repeats monthly!
</Tip>

<Tip>
  **Proposal to Invoice**: Clear proposals prevent billing disputes. Be specific about scope and deliverables.
</Tip>

<Tip>
  **Expense Billback**: Communicate expense policy upfront. Surprise expenses cause client friction.
</Tip>

<Tip>
  **Monthly Close**: Make it routine. Same process every month builds good habits and catches issues early.
</Tip>
